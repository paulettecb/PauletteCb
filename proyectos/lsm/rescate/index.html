    Plan técnico

    Rescate por etapas

    1. Estabilizar UI: mantener esta landing estática como demo publicable.
    2. Separar servicios: aislar detección, gestos, avatar y contenido en módulos probables.
    3. Reconectar ML: cargar modelos solo bajo consentimiento y mostrar estado de permisos.
    4. Validar contenido: revisar cada seña con instructores o comunidad LSM.

    Lo que traía la app

    • Angular + Three.js para avatar GLB.
    • TensorFlow Handpose para puntos de la mano.
    • MoveNet / pose detection para cuerpo.
    • Reglas iniciales para palma abierta, puño y pulgar arriba.
