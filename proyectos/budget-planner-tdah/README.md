# Budget Planner TDAH

Este módulo corre como sitio **estático puro** (HTML/CSS/JS), compatible con GitHub Pages sin build.

## Uso local

Desde la raíz del repo:

```bash
python3 -m http.server 4173
```

Luego abre:

- `http://localhost:4173/proyectos/budget-planner-tdah/index.html`

## Deploy en GitHub Pages

No requiere Vite ni Node para publicar.
