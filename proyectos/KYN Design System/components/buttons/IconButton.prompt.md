Round, hairline icon button — for compact actions like favorite, cart, or close. Always pass an accessible `label`.

```jsx
<IconButton label="Add to favorites"><Heart/></IconButton>
<IconButton label="Cart" size="sm"><ShoppingBag/></IconButton>
```

Use rounded line icons (Lucide / Phosphor) at ~20px. Sizes: `sm` (36px) `md` (44px).
